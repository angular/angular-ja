# ランタイムロケールを手動で設定する

Angularの初期インストールには、すでに米国英語\(`en-US`\)のロケールデータが含まれています。
[`ng build`][AioCliBuild]コマンドで`--localize`オプションを使用すると、[Angular CLI][AioCliMain]は自動的にロケールデータを含み、`LOCALE_ID`の値を設定します。
